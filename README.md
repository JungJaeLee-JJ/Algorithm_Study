# 스터디 방식

- 역할 : **팀장 (신정식) / 총무 (윤여종) / 서기 (장현호) / 깃허브 관리 (이정재) / 블로그 관리(박선영)**
- dp, 그리디, 탐색(bfs, dfs), 구현 , 자유 알고리즘  주제에서 1개씩 정하기 - 총 5문제
- [solved.ac](https://solved.ac/)를 기준으로 Gold 2문제, Siver 3문제로 선정
- 벌금 문제당 1000원, 노쇼 5000, 예치금 20000원
